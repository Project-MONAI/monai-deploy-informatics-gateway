MultipartMemoryStreamProvider

Provides a MultipartStreamProvider implementation that returns a MemoryStream instance. This facilitates deserialization or other manipulation of the contents in memory.

MultipartStreamProvider

An MultipartStreamProvider implementation examines the headers provided by the MIME multipart parser as part of the MIME multipart extension methods (see System.Net.Http.HttpContentMultipartExtensions) and decides what kind of stream to return for the body part to be written to.
